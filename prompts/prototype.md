# Инструкция: Создание прототипа

Ты создаёшь кликабельный прототип продукта. Это НЕ финальный продукт — это демо для проверки идеи.

## Входные данные

Прочитай PRD-файл, который тебе укажут. Там описан продукт и его функции.

## Главные принципы

### Что делаем
- Кликабельный интерфейс, в который можно "потыкать"
- Несколько экранов с навигацией между ними
- Mock-данные прямо в коде (массивы, объекты)
- Красивый, но простой дизайн

### Чего НЕ делаем
- Никаких баз данных
- Никакого бэкенда или API
- Никакой авторизации
- Никаких внешних зависимостей (npm, React и т.д.)

## Технический стек

**Только:**
- HTML
- CSS (встроенные стили или отдельный файл)
- Vanilla JavaScript

**Почему:** Открыл файл в браузере — работает. Без сборки, без установки.

## Структура прототипа

```
prototype/
├── index.html      — главный экран
├── style.css       — общие стили
├── data.js         — mock-данные
└── [другие].html   — дополнительные экраны
```

## Mock-данные

Создай файл `data.js` с тестовыми данными:

```javascript
const mockData = {
  users: [
    { id: 1, name: "Анна", avatar: "👩" },
    { id: 2, name: "Иван", avatar: "👨" },
  ],
  products: [
    { id: 1, title: "Продукт 1", price: 1000 },
    { id: 2, title: "Продукт 2", price: 2000 },
  ]
  // ... данные релевантные для продукта
};
```

## Навигация между экранами

Просто ссылки:
```html
<a href="catalog.html">Каталог</a>
<a href="profile.html">Профиль</a>
```

## Интерактивность

Простой JavaScript для:
- Показать/скрыть элементы
- Переключить состояние (активно/неактивно)
- Отобразить данные из mock

Пример:
```javascript
function showModal() {
  document.getElementById('modal').style.display = 'block';
}
```

## Чек-лист готовности

- [ ] Можно открыть index.html в браузере
- [ ] Все ссылки работают (переходы между экранами)
- [ ] Есть mock-данные, которые отображаются
- [ ] Можно "поклацать" — кнопки реагируют
- [ ] Выглядит как реальный продукт (не wireframe)

## Результат

Создай папку `prototype/` со всеми файлами. После создания скажи:
"Прототип готов. Открой prototype/index.html в браузере."

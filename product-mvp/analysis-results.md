## Философия Апокрифа

> **Апокриф — тихая мастерская для совместного творчества.** Не сообщество, не социальная сеть. Люди приходят играть, а не тусоваться. Анонимность — не фича, а фундамент. Никаких рейтингов, кармы, шлейфа репутации. Чистый лист в каждой новой игре.

### Ключевые принципы
- Площадка для ИГРЫ, не для сообщества
- Анонимность = главное предложение (люди устали от публичности)
- Никакой «кармы», рейтингов, репутации
- Библиотека — не витрина для славы, а «буккроссинг» (оставил — кто-то нашёл)
- Чистота и безопасность без давления публичности

## Решение

Апокриф уже определил свою идентичность — место где тебя не судят. Вопрос не «как вырастить платформу», а «как сделать игру комфортнее».

Три главные боли:
1. **Найти партнёра** — не зная, бросит ли он. Решение: «пробный пост» при отклике
2. **Не потерять игру** — партнёр написал, а ты не узнал. Решение: email-уведомления
3. **Безопасность публикации** — чтобы Библиотека не стала площадкой для гадостей. Решение: буфер 24ч + модерация

## Варианты развития

### Категория А — Улучшение самой игры (приоритет)

1. **Email-уведомления** — партнёр написал → вы узнали. Частота: сразу / раз в час / раз в день / выкл — **следующая задача**
2. **Пробный пост при отклике** — покажи что умеешь писать перед началом игры. Защита без «кармы» — **после email**
3. **Автосохранение черновиков** — localStorage каждые 30с. Закрыл вкладку ≠ потерял пост — **быстро реализовать**
4. **Закладки внутри игры** — пометить важный момент для себя. Навигация в длинных играх — **после пробного поста**

### Категория Б — Безопасность и модерация

5. **Буфер публикации 24 часа** — модератор проверяет до появления в Библиотеке — **перед запуском Библиотеки**
6. **Предпросмотр перед публикацией** — «вот как это будет выглядеть для читателей» — **вместе с буфером**
7. **Система предупреждений** — предупреждение → 2-е → временный бан → постоянный — **до запуска**

### Категория В — Качество жизни

8. **i18n: английский язык** — в 10-20 раз большая аудитория. Стабы уже есть — **когда MVP стабилен**
9. **PWA с push-уведомлениями** — «установить» сайт на телефон — **после email**
10. **Сброс пароля** — сейчас забыл = потерял аккаунт — **до запуска**

### Категория Г — Монетизация (после 300+ игроков)

11. **Подписка «Апокриф+» (3-5$/мес)** — не блокирует игру, даёт больше самовыражения: темы, цвета, экспорт, слоты заявок — **когда есть база**
